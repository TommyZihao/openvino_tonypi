# 基于大模型语音控制的TonyPi人形机器人

会唱跳RAP、搬运货物、巡线避障、打出“丝滑小连招”的人形机器人

让具身智能体Agent理解复杂指令、编排高难动作、玩唱跳RAP、踢花样足球、搬繁重货物

基于幻尔科技TonyPi人形机器人，提示词工程构建Agent智能体，人类只需输入模糊语义语音指令，AI即可自主思考，调用23种原子动作Function Call，编排生成动作列表和函数参数。
