# 基于OpenVINO控制TonyPi人形机器人

基于OpenVINO，在酷睿Ultra的AI PC上，本地部署Qwen2-7B-Instruct开源大模型，语音控制TonyPi人形机器人

让具身智能体Agent理解复杂指令、编排高难动作、玩唱跳RAP、踢花样足球、搬繁重货物
